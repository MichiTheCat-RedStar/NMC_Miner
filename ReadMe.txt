NMC_MINER
=========

NMC_Miner (NimMinerCoins_Miner) - это симулятор майнинга, написанный на языке Nim.
Проект является переосмыслением и улучшением предыдущей работы автора - PMC_Miner (PythonMinerCoins_Miner), которая была реализована на Python.
Теперь код скомпилирован в нативный бинарник, что даёт значительный прирост производительности и позволяет запускать программу на различных платформах без установки интерпретатора.

Автор: MichiTheCat-RedStar
Лицензия: MIT
Версия: release (подробнее написано при запуске или в исходном коде)


ВОЗМОЖНОСТИ
-----------

- Имитация процесса майнинга с генерацией случайного дохода.
- Интерактивная консольная оболочка с набором команд.
- Сохранение и загрузка прогресса в файл save.json.
- Полная информация о балансе, общем заработке и количестве циклов.
- Возможность расширения функционала.


УСТАНОВКА И ЗАПУСК
------------------

Готовые бинарники
В папке bin/ находятся собранные версии для:
- x86_64 (Windows 64)
- i386 (Windows 32)
- Linux_danger (Быстрый, но без обработки ошибок)
- Linux_release (Основная версия)

Скачайте подходящий исполняемый файл и запустите его в терминале или командной строке.

Сборка из исходников
Если у вас установлен компилятор Nim, вы можете собрать проект самостоятельно:

nim c -d:release src/NMC_Miner.nim

Исполняемый файл появится в той же папке, что и исходник.


КОМАНДЫ
-------

После запуска программа ожидает ввод команд. Доступны следующие инструкции:

help, h, ?          - Показать список всех команд
balance, b          - Показать текущий баланс (с двумя знаками после запятой)
info, i             - Вывести расширенную информацию: баланс, общий заработок, количество циклов, средний доход за цикл
start, s            - Запустить один цикл майнинга - генерируется случайное число NMC, которое добавляется к балансу
save, m+            - Сохранить текущий прогресс в файл save.json
load, m-            - Загрузить сохранённый прогресс из файла save.json
exit, e             - Завершить работу программы

Все команды регистронезависимы.


СОХРАНЕНИЕ И ЗАГРУЗКА
---------------------

- Данные сохраняются в формате JSON в файл save.json в той же директории, где запущена программа.
- При загрузке программа проверяет целостность файла; в случае ошибки парсинга создаётся новый файл с нулевыми значениями.
- Если файл отсутствует, прогресс начинается с нуля.


ТЕХНИЧЕСКИЕ ДЕТАЛИ
------------------

- Язык: Nim (статическая типизация, компилируемый в C/C++).
- Используемые модули стандартной библиотеки: rdstdin, unicode, strutils, os, random, json.
- Архитектура: консольное приложение с бесконечным циклом обработки команд.
